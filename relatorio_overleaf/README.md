# Trabalho Final - Métodos Estatísticos 2

## Estrutura do Projeto

Este projeto contém todos os arquivos LaTeX necessários para compilar o trabalho final de Métodos Estatísticos 2 no Overleaf.

### Arquivos Principais

- **main.tex**: Arquivo principal que importa todos os outros
- **capa.tex**: Capa do trabalho
- **introducao.tex**: Introdução
- **descricao_amostra.tex**: Descrição da amostra
- **pergunta1.tex** até **pergunta8.tex**: Análise de cada pergunta de pesquisa
- **conclusoes.tex**: Conclusões gerais
- **referencias.tex**: Seção de referências
- **apendice_codigo.tex**: Código R completo utilizado
- **ref.bib**: Arquivo de referências bibliográficas

### Pasta de Imagens

A pasta `imagens/` deve conter todas as imagens do trabalho. Atualmente contém apenas `unb.png` (logo da UnB).

## Como Usar

1. **Fazer upload no Overleaf**: Você pode fazer upload de todos os arquivos como um projeto novo ou adicionar os arquivos a um projeto existente.

2. **Adicionar as imagens**: 
   - Crie uma pasta chamada `imagens` no Overleaf
   - Faça upload de cada uma das 12 imagens listadas acima com os nomes exatos

3. **Compilar**: Clique em "Recompile" no Overleaf para gerar o PDF

4. **Editar**: Você pode editar qualquer arquivo .tex para fazer ajustes no conteúdo

## Notas Importantes

- O arquivo **apendice_codigo.tex** contém o código R completo utilizado na análise
- O arquivo **ref.bib** contém as referências bibliográficas (você pode adicionar mais se necessário)
- Todos os arquivos estão em português (Brasil)
- O documento segue as normas ABNT para citações
